```
incoming_data/

├── 외통위/
```

**폴더명 규칙**: 한국어 약칭 사용 (`과방위`, `외통위` 등)
**ETL 처리 시**: 폴더명으로 위원회를 판별한다. `manifest_builder.py`의 `FOLDER_TO_COMMITTEE` 매핑 참고.

---

## PDF 파일명 규칙

```
{날짜}_{회차}_{PDF_ID}.pdf
```

예:
```
20240605_1차_123456.pdf
20241015_3차_789012.pdf
```

- `날짜`: `YYYYMMDD` 형식 (confDate 필드에서 추출)
- `회차`: conferNum 필드 (숫자·특수문자 제거 후)
- `PDF_ID`: pdfLinkUrl 쿼리스트링에서 추출한 고유 숫자

같은 날짜 PDF가 여러 개일 수 있다 (소위원회 별도 회의 등). 중복 허용.

## 수집 스크립트

`scripts/crawl_pdfs.py`

**실행 방법**:
```bash
# 전체 위원회
python scripts/crawl_pdfs.py

# 특정 위원회만
python scripts/crawl_pdfs.py 과방위 기재위
```

**주요 설정**:
- `BEGIN_DATE = "20240530"` — 제22대 국회 개원일
- `END_DATE = ""` — 빈 문자열 = 현재까지
- CSRF 토큰: 페이지 GET → `<meta name="_csrf">` 파싱 후 POST 포함
