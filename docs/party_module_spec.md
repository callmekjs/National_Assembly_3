- **위성정당 34명**: 국민의미래 18, 더불어민주연합 16 — **표기는 위성정당 그대로 유지**

- 코드포인트: API 한자는 표준(柳 U+67F3), DB는 호환용(U+F9C9) → **NFKC 정규화 매칭 필수**

## 구성

### 1. `scripts/build_members.py` (수집·적재 — 1회성, 재실행 가능)
- API 수집(브라우저 UA 필수) → `GTELT_ERACO`에 "제22대" 포함 필터
- 최종 당적 = `PLPT_NM` 마지막 항목, 위성정당 포함 표기 그대로
- 원본 보존: `data/members/members_22.json` (재현성)
- 한자 별칭 자동 생성: `data/members/hanja_aliases.json` (한글명↔한자명 쌍 — aliases.py가 로드)
- DB `members` 테이블 DELETE+재삽입 (jsonl_to_postgres 패턴), `db/schema.sql`에도 추가:
  member_id(NAAS_CD PK), name, hanja_name, party(정규화), party_raw(PLPT_NM 원본), era, committees

### 2. `backend/party.py` (순수 판정 함수 + 캐시)
- `party_label(speaker, meeting_date) → "국민의힘(당시 야당)" | "무소속" | None`
- 여야 판정표(상수): `2024-05-30~2025-06-03` 국민의힘 여당 / `2025-06-04~` 더불어민주당 여당 (조기대선·취임일)
- 위성정당은 `SATELLITE_PARENT` 매핑으로 모정당 기준 여야 계산 (표기는 위성정당 그대로 — 예: `국민의미래(당시 여당)`)
- 매칭 키: NFKC(이름)과 NFKC(한자명) 모두 등록 — DB의 호환용 한자 발언자도 매칭
- 서로 다른 정당의 동명이인 → None (틀린 라벨보다 무표기 — 신뢰 원칙)
- members는 첫 사용 시 DB에서 1회 로드해 dict 캐시 (질의마다 조회 없음)
- 맵 구축 로직은 `_build_map(rows)` 순수 함수로 분리 (DB 없이 테스트)

### 3. `answer.py` 주입
- 근거 블록 speaker 줄: `speaker: 엄태영 위원 [국민의힘(당시 야당)]` (party_label 결과가 있을 때만)
- `_source_summary`에 `party` 필드 추가 → 프론트 출처 카드에도 표기 (SourcePanel 소폭 수정)
- `_PARTY_GUARD` 문구 교체: "근거 블록에 [정당] 표기된 것만 사용, 표기 없는 발언자는 추측 금지"
  (기존 3단 방어는 미매칭 발언자용으로 축소·유지)

### 4. aliases.py 확장 (보너스)
- `data/members/hanja_aliases.json` 있으면 로드해 한자 별칭 그룹 자동 등록
  (수동 2명 → 전체 자동. 파일 없으면 기존 정적 사전만 — 하위 호환)

## 발언 자격(role) 게이트 (2026-07-03 추가 — 사용자 규칙)

> 발견: 정동영(의원 겸 통일부장관)이 장관 자격 발언에서 "여당"으로 분류됨.
> 원인: 이름만으로 members 매칭 — 발언의 자격을 보지 않았다.

분류 규칙 (이름이 아니라 **자격** 기준):
- **국회의원 자격**(위원·위원장·소위원장·위원장대리·간사·의원·조정위원장)만 정당·여야 라벨
- **행정부 자격**(장관·차관·청장·처장·차장·총장·국무위원·대통령실·비서관·대변인·협력관·
  정부위원·행정기관 위원장) → **"정부측"** — members 에 있는 겸직 의원이라도.
  `…위원장$` 패턴은 국회 위원장과 겹쳐 보이지만 국회 쪽은 exact 집합이 먼저 걸러냄
- **후보자**(장관후보자·위원장후보자 등 `…후보자?$`) → 무표기, 직함 그대로
  (아직 행정부 소속이 아님 — 2026-07-03 사용자 규칙 2차)
- **증인·참고인·진술인** → 정당 무표기 (출석 지위는 role 로 이미 표기).
  프롬프트가 "출석 지위 그대로 분류, 소속 기관이 근거 본문에 있으면 '○○ 증인' 병기"를 지시
- **국회 스태프**(전문위원·수석전문위원·입법조사관 등) → 무표기 ("수석" 등 행정부 패턴 오폭 방지)
- **미상 role** → 무표기 (확신 없으면 라벨을 달지 않는다)

검증 실측: 정동영 통일부장관후보자→정부측, 김석기 위원장→국민의힘(당시 야당) 유지,
최혁진→더불어민주연합(당시 여당) (위성정당 표기+모정당 여야 동시 확인), 티몬 대표이사→무표기
